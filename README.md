# DQN-pytorch-Atari
Implement DQN and DDQN algorithm on Atari gamesï¼Œsuch as BreakoutNoFrameskip-v4, PongNoFrameskip-v4,BoxingNoFrameskip-v4.


## Requirements
- cuda 10.1
- python 3.7.6
- gym 0.17.6
- pytorch 1.5.0
- tdqm



## Usage Example
```
$ python main.py --env_name BreakoutNoFrameskip-v4-v2
```
